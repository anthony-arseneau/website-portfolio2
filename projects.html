Self-Leveling Platform

Built by team members and I, the Balancing Automated Leveraging and Levelling Electronic Robot (B.A.L.L.E.R.), as is its name suggests, balances a 1 inch steel bearing ball on its surface through any disturbance. This project was helped with open source files provided by Aaed Musa. About 40 hours were needed to complete this project using an Arduino with C++ code.

Self-Hosted Website

The website you are currently on is hosted by my own Raspberry Pi 5 at home. The learning journey for this process was quite challenging, but was rewarding at the end. An unexpected time consumming aspect of the project was the web page development using HTML, CSS, and Javascript. Attention to detail is needed to make sure everything is in its place in any scenario. After learning from many mistakes, creating a new website now would take approximately half the time it took to make this one. I also implemented automation features using crontab, such as, IP updating and website updating when pushing files to GitHub.
