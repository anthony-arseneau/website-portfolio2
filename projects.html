Self-Leveling Platform

Designed and developed by my team and me, the Balancing Automated Leveraging and Levelling Electronic Robot (BALLER) is an advanced mechatronic system engineered to dynamically stabilize a 1-inch steel bearing ball on its surface, compensating for external disturbances through precise PID control algorithms. The project involved extensive use of 3D CAD software for the mechanical design and optimization, and key components were fabricated using 3D printing for structural accuracy. Electrical analysis was conducted to ensure proper power distribution and signal integrity throughout the system. Approximately 50 hours were dedicated to the project, leveraging an Arduino microcontroller programmed in C++ for real-time feedback control. Open-source resources provided by Aaed Musa were incorporated to assist with system integration and development.

Encrypted Private Group Chat

Developed a secure messaging application that facilitates private group chat functionality through advanced encryption techniques. This project employed RSA for asymmetric encryption and AES for symmetric encryption to ensure data confidentiality, efficiency and integrity. The application also utilized SHA-256 for hashing, providing an additional layer of security for user credentials.

Leveraging Java and socket programming, I designed and implemented a robust client-server architecture, enabling real-time communication while maintaining high security standards. Through this project, I gained practical experience in applying cryptographic algorithms and networking frameworks, further enhancing my understanding of secure data transmission and application design.

Self-Hosted Website

The website you are currently viewing is hosted on a server that I personally set up and configured. The learning curve for setting up and managing the server infrastructure was steep but ultimately rewarding. An unforeseen but time-consuming aspect of the project was the front-end development, which required the use of HTML, CSS, and JavaScript. Ensuring the site maintained consistent functionality and visual integrity across various browsers and devices demanded a high level of attention to detail. Through multiple iterations and the resolution of numerous technical challenges, I have significantly streamlined my workflow; creating a new website would now only take a fraction of the time compared to the initial effort.

To enhance system efficiency, I implemented automation features using crontab, including automatic IP updates and seamless website updates whenever changes are made. Additionally, a proper cooling system was integrated into the server setup to ensure optimal thermal management, safeguarding performance and stability during long periods of operation under load.
